# ⚔️ AION 2 幣值換算最佳化工具 (AION 2 Currency Exchange Optimizer)

專為《AION 2》玩家打造的單頁面網頁工具，用於精準計算新台幣 (NTD) 轉換為遊戲幣 (基納) 的最高效益途徑。本工具會自動對比「官方/代儲寶石」與「第三方幣商」的真實匯率，並扣除所有隱藏手續費。

## ✨ 功能特色 (Features)

* 📊 **多管道收益比較**：一鍵比較「兌換所 1000 萬箱」、「兌換所 200 萬箱」與「第三方幣商 (交易所拉裝)」的最終實拿基納。
* 🧮 **精準稅金扣除**：完整內建交易所雙重手續費 (預設為上架 2% + 交易完成 10%)，並支援自訂第三方交易平台 (如 8591) 的額外抽成比例，讓隱藏成本無所遁形。
* 💾 **自動記憶設定**：輸入過的各項浮動數值 (如代儲物價、幣商匯率) 會透過瀏覽器的 Local Storage 自動儲存，下次開啟網頁無需重新輸入。
* 📱 **響應式網頁設計 (RWD)**：介面簡潔，完美支援電腦與智慧型手機操作，隨時隨地皆可計算。

## ⚠️ 免責聲明與風險提示 (Disclaimer)

本工具內含之「一般幣商收購 (拉裝備)」計算選項，涵蓋了現金交易 (RMT, Real Money Trading) 的情境。
**強烈提醒：** 透過第三方平台進行現金交易，並利用遊戲內交易所進行高價拉取裝備洗錢，屬於違反多數遊戲官方服務條款之行為。經官方查緝可能導致帳號遭到凍結或永久停權。
> 本專案僅提供數學模型與數值轉換參考，不鼓勵、不推廣任何違反遊戲規章之行為。玩家請自行評估並承擔帳號安全風險。

## 🛠️ 計算邏輯說明 (How it Works)

1.  **寶石成本單價**：`花費台幣 ÷ 獲得寶石數量 = 單顆寶石台幣成本`
2.  **寶箱收益**：`(總預算 ÷ 單顆寶石台幣成本) ÷ 寶箱需求寶石數 × 寶箱基納面額`
3.  **幣商收購實拿**：`總預算 × 幣商匯率 × (1 - (上架手續費% + 交易手續費% + 額外抽成%))`
系統會自動將上述三者結果進行排序，並將實拿收益最高者標記為 **[最划算]**。
